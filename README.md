# OpenRouter Runner

OpenRouter Runner is a monolith inference engine, built with [Modal](https://modal.com/), used for lots of the open source models hosted in a fallback capacity on [openrouter.ai](https://openrouter.ai).

## Engines

- vLLM
- HF Transformers
